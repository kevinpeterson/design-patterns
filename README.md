design-patterns
===============
Example Design Patterns in Java.

Setup
-----
Import into Eclipse, Intellij, or whatnot.
This is a Maven project, but there are no dependencies set.

Patterns
--------
These are the patterns to focus on:
* Factory Method
* Builder
* Prototype
* Flyweight
* Memento
* Proxy
* Visitor
* Template Method
* Mediator
* Command
* State
* Factories
* Singleton
* Object Pool
